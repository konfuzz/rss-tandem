# Дата: 2026-02-21

**Что было сделано:** начали с обсуждения концепции командного проекта. В итоге договорились, что будем делать аппку для проверки знаний по JS и подготовки к собесам, что-то похожее на виджет-тренажер, как предлагается в примерах таска. Основные модули: лендинг, дашборд и набор интерактивных виджетов.

Предложенные идеи виджетов: Квиз — классические вопросы с вариантами ответов и таймером. ИИ-интервьювер — чат с AI, который оценивает полноту ответа и дает фидбек. CSS Battle — верстка по картинке «на глаз» с попиксельным сравнением. Конструктор JS-методов — аналог RSS-Puzzle, где нужно собирать синтаксис методов перетаскиванием слов. CSS-специфичность — выбор приоритетных правил. Самый эффективный код — сравнение производительности фрагментов. Поиск ошибки — указание строки, где код упадет.

Сделали прототип CSS Battle, чтобы проверить, реализуемая это идея или нет. Выяснилось, что технически это реализуемо (сравнение через canvas), но встречаются баги. Если решим включать этот прототип в проект, нужно его доработать и тщательно потестить.

Обсудили технологический стек. Пока остановились на Vue.js для клиента. Никто из нас пока не знаком с этим фреймворком и его экосистемой, но все в команде готовы пробовать.

**Мысли / Планы:** пока на этапе active brainstorming: есть концепция, есть пул идей, есть прототип одного из виджетов, есть репо. Нужно поизучать Vue.js, подумать над стеком для бэка, настроить репо и пайплайн.
