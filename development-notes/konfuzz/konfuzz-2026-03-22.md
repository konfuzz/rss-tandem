Сегодня проделал большой кусок работы:
- Настройка навигации: Спроектировал структуру маршрутов с помощью vue-router. Создал базовые компоненты-заглушки (Views) для всех ключевых разделов, что позволило сразу протестировать переходы.
- Интеграция Auth-системы: Реализовал полноценные страницы регистрации и логина. Настроил взаимодействие с API через fetch, включая обработку ошибок валидации со стороны бэкенда.

Из сложностей столкнулся с тем, что нам нужно где-то хранить состояние авторизации/токен, а localstorage не реактивен и его пришлось бы постоянно дёргать при каждой операции. Выглядит неэффективно и неудобно. В результате в проект заехала Pinia и теперь состояние авторизации синхронизируется между всеми компонентами, а дальнейшая разработка несколько упростится благодаря тому, что теперь есть централизованный источник правды и не нужно заниматься прокидыванием пропсов через компоненты.
