# Дата: 2026-03-27

- Улучшение сохранения состояния квиза между перезагрузками.
- Добавление полноценного post-quiz summary экрана.

## Что сделано

- Переработан store квиза:
  - добавлена защита от повторной записи одного и того же ответа по `questionId`
- Переработан экран квиза:
  - добавлена нормальная инициализация незавершённого квиза из persisted state
  - после завершения попытки пользователю показывается summary вместо мгновенного сброса
- Обновлён `WidgetWrapper`:
  - сбор payload для `/quiz/finish` вынесен в общие util-функции
  - итоговая статистика по попытке считается централизованно
- Добавлен новый `QuizSummaryCard`:
  - карточка с итогом прохождения
  - прогресс-бар по набранным баллам
  - блоки со сводной статистикой
  - теги со средними баллами по категориям
  - действия "Новый квиз" и "На дашборд"
- Вынесена логика расчёта результатов в отдельный util:
  - расчёт `totalScore`
  - расчёт `totalDuration`
  - расчёт `averageScore`
  - расчёт количества идеальных ответов
  - расчёт средних значений по категориям
- При logout теперь дополнительно очищается состояние квиза, чтобы не тянуть старую попытку между сессиями.

## Итого

- Квиз стал лучше переживать перезагрузку страницы и возврат пользователя в приложение.
- После финиша появился внятный UX со статистикой, а не просто сброс состояния.
