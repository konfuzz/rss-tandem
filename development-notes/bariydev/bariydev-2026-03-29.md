# Дата: 2026-03-29

## Исправление ошибки при потере связи с сервером

- Во всех виджетах (Poll, MultipleChoice, CodeAnalysis, MethodBuilder) при сетевой ошибке catch-блок ставил `status = 'fail'` и вызывал `emit('validated', true)`.
- Из-за этого виджет показывал "Ответ неверный", хотя ответ даже не дошёл до сервера.
- Теперь при ошибке сети статус остаётся `playing`, показывается уведомление "Не удалось проверить ответ. Попробуйте ещё раз.", и пользователь может повторить отправку.

## Переключение между своим и правильным ответом

- Во всех виджетах добавлено сохранение состояния пользовательского ответа перед показом правильного.
- После просмотра правильного ответа кнопка меняется на "Вернуться к моему ответу" — восстанавливает исходный выбор пользователя.

## Улучшения MethodBuilder

- Добавлена проверка перед отправкой.
- Предупреждающие уведомления автоматически сбрасываются при взаимодействии с конструктором (клик, drop, перетаскивание).

## Улучшение layout WidgetWrapper

- Установлено ограничение максимальной ширины карточки.
